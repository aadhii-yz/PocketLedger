## What's inside

<CardGrid stagger>
  <Card title="Point of Sale" icon="laptop">
    Fast barcode-based billing with camera scanning, multiple payment methods, and instant thermal receipt printing.
  </Card>
  <Card title="Multi-location Inventory" icon="list-format">
    Track stock across warehouses and shops. Low-stock alerts, stock movements audit log, and inter-location transfers.
  </Card>
  <Card title="Wireless Printing" icon="forward-slash">
    Companion app bridges the cloud backend to your LAN printers. Supports USB and TCP/IP — auto-detected, no IP entry needed.
  </Card>
  <Card title="Offline-ready PWA" icon="puzzle">
    Installable on Android, Windows, and Linux. Core reference data cached locally; checkout blocked when offline to prevent overselling.
  </Card>
  <Card title="Role-based Access" icon="approve-check-circle">
    Four roles: Admin, Manager, Stock Entry, and POS (Billing). POS users are scoped to a single shop automatically.
  </Card>
  <Card title="AI-friendly Docs" icon="open-book">
    Every page has an Ask AI box. Type your question, pick Claude/ChatGPT/Gemini/Perplexity, and get an answer with full documentation context.
  </Card>
</CardGrid>
